# Тетрис

Классическая игра Тетрис на HTML/CSS/JS.

## Управление

- ← → — движение
- ↑ — поворот
- ↓ — ускорить
- Пробел — сразу вниз

## Как играть

Заполняй горизонтальные линии, чтобы получать очки. Чем выше уровень — тем быстрее падают фигуры.

## Запуск

Открой файл `index.html» в любом браузере.

## Файлы

- `index.html` — разметка
- `style.css` — стили
- `tetris.js` — логика игры
